# Nes2C64
16F84 based Nes to C64 gamepad adapter
---------------------------------------------------------------
A simple adapter to use a NES gamepad as C64 Joystick. 
'A' button mapped as FIRE
'B' button mapped as 'UP'
Eventually, 'SELECT' or 'START' may be used as space, connecting 
to C64's PORT2 fire
